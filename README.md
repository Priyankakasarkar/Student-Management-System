Student Management System | Java Full Stack Project

Developed a Student Management System web application to efficiently manage student records and academic information. The system allows users to add, update, view, and delete student details through an interactive interface. It simplifies administrative tasks by organizing student data in a structured database.

The application was built using Java, Spring Boot, Hibernate, MySQL, HTML, CSS, and following the MVC architecture to ensure clean code structure and scalability. RESTful APIs were implemented for seamless communication between the frontend and backend.

Key Features:

Add, update, delete, and view student records

User-friendly interface for easy data management

Secure backend using Spring Boot and Hibernate

MySQL database integration for efficient data storage

Technologies Used:
Java | Spring Boot | Hibernate  | MySQL | HTML | CSS | REST API | Maven
